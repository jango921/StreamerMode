# StreamerMode

StreamerMode is a plugin that allows you to stream without revealing any information

## Reasoning
Primarily making this to stream evercold and I want to be able to use plugins and stream.

## Currently supported
* Changes username on nameplate and chat!
* Changes world you are connected to.

## Plans
Spoof name in different ui (ex. in freecompany list / party members / any revealing locations locally)
